## Review Checklist

### **Code**
- Clean and readable  
- Follows project style  
- No unused or duplicate code  

### **Functionality**
- Works as intended  
- No breaking changes  

### **Testing**
- Tests added/updated  
- All tests pass  

### **Documentation**
- Clear comments  
- Docs/screens updated (if needed)  

### **Security**
- No secrets exposed  
- Input validation considered  

### **Final Check**
- PR title + description clear  
- Commits clean  
- Ready to merge
